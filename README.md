# Actioner
The Actioner is a c++ project that initiates or activates webhooks in response to events generated by an external third-party application.

# Eevnt data formate 
```json
{
  "event_name": {
    "event_data_key_1": "a",
    "event_data_key_2": "b"
  }
}
```
